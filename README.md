**Target Filter Presets**

A BepInEx mod for Nuclear Option that lets you save and switch target-selection filter presets with keybinds.

Features:

- Save target filter presets for the in-game target selection menu
- Apply presets instantly with keybinds
- Cycle next/previous preset with keybinds
- HUD-style popup showing the active preset name
- Rename presets
- Add preset slots up to a 20-preset cap
- Click-to-bind keybind controls in the NO UI Framework menu
- Edit filters from the mod menu
- Starts with one preset by default so players only add what they need

Requires:

- BepInEx
- NO UI Framework

Useful for quickly switching between target priorities, such as air targets, ground vehicles, ships, SAMs, or custom mission-specific filter setups.
